Frameworks CSS + Bootstrap

CDN CSS-->
        es un package en la cual la info. se llama desde la web (está en la nube) sin necesidad de descargar los package de bootstrap. 
        Hay páginas construídas con CDN.
        Generalmente, se utiliza para hacer proyectos rápidos o probar ciertas
        cosas, dado que solo se hace un link pequeño en el css para llamar al
        package.

        El link CSS de CDN tiene que ir siempre arriba de nuestro CSS (Main).
        Esto es, porque que nuestro CSS esté despues permite que nuestro cambio
        (en nuestro CSS) sea el que valga, pudiendo pisar el del CSS CDN.

        Ej:<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">

CDN Javascript
    --> muchos componentes de bootstrap utilizan javascript. Es decir, necesitan
    componentes o plugin de javascript.

    El plugin de javascript debe ir justo antes de la etiqueta final del body.
    (</body)

    Ej:
    <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.10.2/dist/umd/popper.min.js" integrity="sha384-7+zCNj/IqJ95wo16oMtfsKbZ9ccEh31eOz1HGyDuCQ6wgnyJNSYdrPa03rtR1zdB" crossorigin="anonymous"></script>

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.min.js" integrity="sha384-QJHtvGhmr9XOIpI6YVutG+2QOK9T+ZnN4kzFN1RtK3zEFEIsxhlmWl5/YESvpZ13" crossorigin="anonymous"></script>

Agregar Componentes -->
    En la página de Bootstrap, sección Docs, en la barra lateral aparece "Components". Desde ahí puedes saber cómo llamar a los diferentes 
    elementos que conformarán tu página, como: navbar, carrusel, etc.
